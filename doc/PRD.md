# LensMind — 电商视频自动生成 Agent PRD

## 1. 产品概述

**LensMind** 是一个专注于电商产品视频的 AI Agent 创作平台。用户输入产品名称或上传产品图片后，AI 自动完成：**产品分析 → 剧本/分镜脚本 → 模特图生成 → 场景图生成 → 分镜片段生成 → 视频合成 + 配音**。目标是像 OiiOii 一样，把复杂的 AI 视频创作变成"自动挡"。

## 2. 目标用户

- 电商卖家（淘宝、抖音小店、Shopify）
- 代运营 / MCN 机构
- 品牌营销团队

## 3. 核心流程

```
输入（产品名称 / 产品图）
        │
        ▼
  ┌──────────┐
  │ 产品分析  │  AI 提取卖点、风格、目标人群
  └────┬─────┘
       │
       ▼
  ┌──────────┐
  │ 剧本生成  │  LLM 生成口播文案 + 分镜脚本
  └────┬─────┘
       │
       ▼
  ┌──────────────────┐
  │ 模特图生成        │  AI 生成产品展示模特图
  │ 场景图生成        │  AI 生成对应场景背景
  └──────┬───────────┘
         │
         ▼
  ┌──────────┐
  │ 分镜片段  │  模特图 + 场景图 → 逐镜视频片段
  │ 生成      │  (图生视频 / 运镜动画)
  └────┬─────┘
       │
       ▼
  ┌──────────┐
  │ 视频合成  │  拼接片段 + TTS 配音 + 字幕 + BGM
  └────┬─────┘
       │
       ▼
    最终视频
```

### 3.1 输入方式

| 方式 | 说明 |
|------|------|
| 产品名称 | 输入商品名，AI 自动分析 |
| 上传产品图 | 上传 1-5 张产品图片 |
| 产品名 + 产品图 | 两者结合，效果更好 |

### 3.2 各阶段说明

#### 阶段 1：产品分析
- 输入产品名称/图片 → AI 输出：品类、目标人群、核心卖点、视觉风格建议
- 非单纯文字分析，需要结合图像理解产品外观

#### 阶段 2：剧本生成
- LLM 根据产品分析结果生成：
  - 视频标题
  - 口播文案（分句）
  - 分镜脚本（每个分镜的画面描述、运镜方式、时长）
  - 整体风格/语调

#### 阶段 3：模特图 + 场景图生成
- **模特图**：AI 生成模特展示产品的图片（保持模特一致性）
- **场景图**：AI 生成每个分镜需要的背景场景
- 两者可并行生成

#### 阶段 4：分镜片段生成
- 每个分镜 = 模特图 + 场景图 → 图生视频/运镜动画
- 输出一个个独立的视频片段

#### 阶段 5：视频合成
- 拼接所有分镜片段
- 叠加 TTS 配音（对应每段口播文案）
- 添加字幕
- 添加背景音乐
- 输出最终视频

## 4. 功能范围

### MVP（当前阶段）

| 功能 | 说明 |
|------|------|
| 产品名称输入 | 输入商品名称，AI 分析并生成视频 |
| 产品图片上传 | 上传产品图辅助分析 |
| 产品分析 | LLM 分析卖点、人群、风格 |
| 剧本生成 | 生成分镜脚本 + 口播文案 |
| 模特图生成 | AI 生成模特展示图（调 API） |
| 场景图生成 | AI 生成场景背景（调 API） |
| 分镜片段生成 | 图生视频生成每个分镜片段（调 API） |
| TTS 配音 | 文案转语音 |
| 字幕生成 | 音频对齐生成字幕 |
| 视频合成 | 片段拼接 + 配音 + 字幕 + BGM |

### 后续版本

| 功能 | 说明 |
|------|------|
| **智能画布** | 可视化编排分镜，拖拽调整，AI 对话修改 |
| 拉片复刻 | 上传参考视频，自动拆解分镜并替换产品 |
| 资产库 | 角色/场景/运镜保存为资产可复用 |
| 批量生成 | 一次生成多条不同风格视频 |
| 模板/Skill 市场 | 预设行业模板，一键套用 |
| 人工审核节点 | 每阶段可插入人工确认 |

## 5. 技术依赖

| 能力 | MVP 技术方案 |
|------|-------------|
| LLM 文案生成 | OpenAI GPT-4o / Anthropic Claude |
| 模特图生成 | Stable Diffusion / Flux / 可灵 API |
| 场景图生成 | Stable Diffusion / Flux / 可灵 API |
| 图生视频 | Runway Gen-3 / 可灵 / Sora API |
| TTS 配音 | OpenAI TTS / Azure Speech / 火山引擎 |
| ASR 字幕 | Whisper |
| 视频合成 | FFmpeg |

## 6. 非功能性需求

| 维度 | 要求 |
|------|------|
| 单视频耗时 | MVP 阶段 < 10 分钟 |
| 图片质量 | 模特图和场景图需满足电商广告标准 |
| 角色一致性 | 同一视频中模特外观保持一致 |
| 可扩展性 | 后续可接入画布编辑器 |

## 7. 竞品参考

**OiiOii (oiioii.tv)**：7 大 AI Agent 协作的视频创作平台，核心功能包括智能画布（自然语言交互式编辑）、拉片复刻（上传爆款视频自动拆解分镜并元素替换）、Skill 库（预设行业工作流）。已接入 Seedance 2.0 等视频模型，支持 2D/3D/CG/真人多种画风。
